# Captures d'écran du TP Big Data

Ce dossier contient les captures des interfaces web du cluster :

## 1. hadoop.png
Interface web de **Hadoop HDFS NameNode**
- Accès : http://localhost:9870
- Montre l'état du système de fichiers distribué
- Affiche les DataNodes connectés et l'espace disponible

## 2. Yarn.png  
Interface web de **YARN ResourceManager**
- Accès : http://localhost:8088
- Gestion des ressources CPU/mémoire
- Liste des applications en cours d'exécution
- Vue des NodeManagers disponibles

## 3. Spark.png
Interface web de **Spark Master**
- Accès : http://localhost:8080
- Montre les workers connectés au cluster
- Affiche les applications Spark en cours
- Permet de suivre l'exécution des jobs
